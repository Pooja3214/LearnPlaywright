# Chapter 09 — User Input

Taking user input via browser `prompt`, Node.js `readline`, and `prompt-sync`.

## Files

| File | Description |
|------|-------------|
| `68_User_Input.js` | Browser-based input using `prompt` |
| `69_Node_readline.js` | Node.js `readline` module for CLI input |
| `70_prompt_sync.js` | Synchronous user input with `prompt-sync` |

## How to Run

### Browser-based (`68_User_Input.js`)
Open in browser console or an HTML file — `prompt` is not available in Node.js.

### Node.js `readline` (`69_Node_readline.js`)
```bash
node 69_Node_readline.js
```

### `prompt-sync` (`70_prompt_sync.js`)
```bash
node 70_prompt_sync.js
```

> Make sure `prompt-sync` is installed:
> ```bash
> npm install prompt-sync
> ```
