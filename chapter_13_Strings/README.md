# Chapter 13 — Strings

String creation, properties, searching, substring extraction, transformation, and conversion methods.

## Files

| File | Description |
|------|-------------|
| `118_Strings.js` | String basics: quotes, template literals, multiline strings, and String interface |
| `119_String_Properties.js` | String properties: `length`, index access, `charAt()`, `charCodeAt()` |
| `120_Searching_Strings.js` | Searching methods: `includes()`, `startsWith()`, `endsWith()`, `indexOf()`, `lastIndexOf()`, `search()` |
| `121_Substring.js` | Substring extraction: `slice()`, `substring()`, `at()` |
| `122_Transform.js` | String transformations: `toUpperCase()`, `toLowerCase()`, `trim()`, `replace()`, `replaceAll()`, `split()`, `join()` |
| `123_String_Conversion.js` | Type conversion: `toString()`, `Number()`, `parseInt()`, `parseFloat()` |

## How to Run

```bash
node 118_Strings.js
```
